# Mero Accounting

A professional accounting system integrated into the Mero Jugx platform. Inspired by Liberu Accounting.

## Overview
Mero Accounting provides double-entry accounting, invoicing, and financial reporting for organizations.

## Features
- **Banking**: Accounts, transactions, and reconciliations
- **Sales**: Invoices, revenues, and customers
- **Purchases**: Bills, payments, and vendors
- **Double-Entry Accounting**: Journal entries and chart of accounts
- **Reports**: Balance sheet, P&L, tax reports

## Usage
Access via the Mero Jugx marketplace or your organization dashboard.
