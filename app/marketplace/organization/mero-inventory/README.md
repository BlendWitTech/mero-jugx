# Mero Inventory

A professional inventory management system integrated into the Mero Jugx platform. Inspired by Liberu Inventory.

## Overview
Mero Inventory provides complete stock management, multi-warehouse support, and order processing capabilities for organizations.

## Features
- **Product Management**: Variants, categories, and attributes
- **Multi-Warehouse**: Manage stock across multiple locations
- **Stock Control**: Transfers, adjustments, and stocktakes
- **Purchase Orders**: Manage suppliers and incoming stock
- **Sales Orders**: Process client orders and shipments

## Usage
Access via the Mero Jugx marketplace or your organization dashboard.
