# Creator Portal

The dedicated dashboard and authentication portal for Creator accounts on the Mero Jugx platform.

## Overview
Provides a simplified, creator-focused interface for managing content, earnings, and profile features.

## Features
- **Dedicated Login/Register**: Simplified onboarding for creators
- **Creator Dashboard**: Key metrics and quick actions
- **Organization Conversion**: Ability to upgrade to a full Organization account

## Architecture
- **Backend**: `CreatorPortalModule`
- **Frontend**: `CreatorPortalRouter` handling `/creator/*` routes
