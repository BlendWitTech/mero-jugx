Getting Started

Creating an Account

  1. Visit the registration page
  2. Fill in your details:
    • Full Name
    • Email Address
    • Password (minimum 8 characters)
  3. Click "Register"
  4. Check your email for verification link
  5. Click the verification link to activate your account

Logging In

  1. Go to the login page
  2. Enter your email and password
  3. Click "Login"
Two-Factor Authentication: If you have enabled 2FA, after entering your email and password, you'll need to enter the 6-digit code from your authenticator app and click "Verify".

First Login

On your first login, you'll be guided through:

  • Profile setup
  • Organization selection or creation
  • Notification preferences

Account Management

Profile Settings

Access your profile from the top-right menu.

Edit Profile:

  1. Click on your profile picture or name
  2. Select "Profile Settings"
  3. Update your information:
    • Full Name
    • Profile Picture
    • Phone Number
    • Bio

Change Password:

  1. Go to Profile Settings
  2. Click "Security"
  3. Click "Change Password"
  4. Enter current password
  5. Enter new password (minimum 8 characters)
  6. Confirm new password
  7. Click "Update Password"

Two-Factor Authentication (2FA)

Enable 2FA for enhanced security:

  1. Go to Profile Settings > Security
  2. Click "Enable Two-Factor Authentication"
  3. Scan the QR code with your authenticator app (Google Authenticator, Authy, etc.)
  4. Enter the 6-digit code to verify
  5. Save your backup codes in a safe place

Disable 2FA:

  1. Go to Profile Settings > Security
  2. Click "Disable Two-Factor Authentication"
  3. Enter your password to confirm

Organizations

Understanding Organizations

Organizations are workspaces where teams collaborate. You can:

  • Belong to multiple organizations
  • Have different roles in each organization
  • Switch between organizations easily

Joining an Organization

By Invitation:

  1. Check your email for an invitation
  2. Click the invitation link
  3. Log in if needed
  4. Accept the invitation
  5. You'll be added to the organization

By Code:

  1. Get the organization join code from an admin
  2. Go to Organizations page
  3. Click "Join Organization"
  4. Enter the join code
  5. Click "Join"

Creating an Organization

  1. Go to Organizations page
  2. Click "Create Organization"
  3. Fill in the details:
    • Organization Name
    • Description (optional)
    • Plan/Package (if applicable)
  4. Click "Create"
  5. You'll automatically become the owner

Switching Organizations

  1. Click the organization selector in the top navigation
  2. Select the organization you want to switch to
  3. The interface will update to show that organization's data

Leaving an Organization

  1. Go to Organizations page
  2. Find the organization you want to leave
  3. Click "Leave Organization"
  4. Confirm your decision
Note: If you're the owner, you must transfer ownership first.

Chat & Communication

Starting a Chat

Direct Messages:

  1. Go to Chat section
  2. Click "New Chat"
  3. Search for a user
  4. Click on their name to start chatting

Group Chats:

  1. Go to Chat section
  2. Click "New Group"
  3. Enter group name
  4. Add members
  5. Click "Create Group"

Sending Messages

  • Type your message in the input field
  • Press Enter to send
  • Use Shift+Enter for a new line

Message Features:

  • @Mentions: Type @ to mention a user
  • Emojis: Click the emoji button or type :emoji_name:
  • File Attachments: Click the attachment icon
  • Voice Messages: Hold the microphone button to record

Making Calls

Voice/Video Calls:

  1. Open a chat with a user
  2. Click the call button (phone or video icon)
  3. Wait for the other user to accept
  4. Grant microphone/camera permissions when prompted

During a Call:

  • Mute/unmute: Click the microphone icon
  • Turn video on/off: Click the camera icon
  • End call: Click the end call button

Chat Settings

Access chat settings from the chat header:

  • Change chat name (for groups)
  • Add/remove members (for groups)
  • Leave chat
  • Mute notifications
  • Search messages

Multi-Factor Authentication (MFA)

Setting Up MFA

Enable MFA for enhanced account security using an authenticator app.

Check MFA Status API

GET /mfa/check
Authorization: Bearer <token>

What it does: Returns whether MFA is enabled and setup status.

Initialize MFA Setup API

POST /mfa/setup/initialize
Authorization: Bearer <token>

Response: Returns QR code data URL and secret for authenticator app.

What it does: Generates TOTP secret and QR code for setup.

Verify MFA Setup API

POST /mfa/setup/verify
Authorization: Bearer <token>
Content-Type: application/json

{
  "code": "123456"
}

What it does: Verifies TOTP code and completes MFA setup, generates backup codes.

Get Backup Codes API

GET /mfa/backup-codes
Authorization: Bearer <token>

What it does: Returns MFA backup codes (one-time use for account recovery).

Regenerate Backup Codes API

POST /mfa/backup-codes/regenerate
Authorization: Bearer <token>
Content-Type: application/json

{
  "code": "123456"
}

What it does: Regenerates new backup codes after verifying current TOTP code.

Disable MFA API

DELETE /mfa/disable
Authorization: Bearer <token>
Content-Type: application/json

{
  "code": "123456"
}

What it does: Disables MFA after verifying TOTP code.

Using MFA

  1. Download an authenticator app (Google Authenticator, Authy, Microsoft Authenticator)
  2. Initialize MFA setup to get QR code
  3. Scan QR code with your authenticator app
  4. Enter 6-digit code to verify
  5. Save backup codes in a safe place

Troubleshooting

Can't Log In

Forgot Password:

  1. Click "Forgot Password" on login page
  2. Enter your email
  3. Check email for reset link
  4. Follow instructions to reset password

Account Locked:

  • Too many failed login attempts
  • Wait 15 minutes or contact support

Can't Send Messages

Check:

  • Internet connection
  • You're in the correct organization
  • Recipient hasn't blocked you
  • Organization chat permissions

Can't Join Organization

Common Issues:

  • Invalid join code
  • Organization is full
  • You've been blocked
  • Organization doesn't exist

Solution:

  • Verify the join code
  • Contact the organization admin

Documents Not Uploading

Check:

  • File size (max 50MB)
  • File format is supported
  • You have upload permissions
  • Internet connection is stable

Try:

  • Compress large files
  • Convert to supported format
  • Check available storage space

Not Receiving Notifications

Check Settings:

  1. Go to Settings > Notifications
  2. Verify notifications are enabled
  3. Check browser permissions for push notifications

Browser Settings:

  • Chrome: Settings > Privacy > Notifications
  • Firefox: Settings > Privacy > Permissions
  • Safari: Preferences > Websites > Notifications

Keyboard Shortcuts

  • Ctrl/Cmd + K: Search
  • Ctrl/Cmd + /: Show all shortcuts
  • Ctrl/Cmd + N: New chat/document
  • Escape: Close modals/panels

Complete User API Reference

Authentication APIs

Endpoint Method Description
/auth/organization/register POST Register new organization and account
/auth/login POST Login to account
/auth/refresh POST Refresh access token
/auth/verify-email GET Verify email address
/auth/forgot-password POST Request password reset
/auth/reset-password POST Reset password
/auth/logout POST Logout from account

User Profile APIs

Endpoint Method Description
/users/me GET Get current user profile
/users/me PUT Update user profile
/users/me/change-password PUT Change password
/users/me/download-data GET Download all account data (GDPR)

Organization APIs (User Context)

Endpoint Method Description
/organizations GET List user's organizations
/organizations/me GET Get current organization
/organizations/switch PUT Switch to different organization

Chat & Communication APIs

Endpoint Method Description
/chats GET List all user's chats
/chats POST Create new chat
/chats/:id GET Get chat details
/chats/:id/messages GET Get chat messages
/chats/:id/messages POST Send message
/calls/chats/:chatId POST Start call

Notification APIs

Endpoint Method Description

/notifications/unread-count GET Get unread count
/notifications/:id/read PUT Mark as read
/notifications/read-all PUT Mark all as read
/notifications/preferences GET Get notification preferences

MFA APIs

Endpoint Method Description
