# Backend Shared Code

This directory contains backend-specific shared code.

## Structure

- `utils/` - Backend utility functions
- `types/` - Backend TypeScript types
- `constants/` - Backend constants
- `decorators/` - Custom NestJS decorators
- `guards/` - Custom NestJS guards

## Usage

```typescript
import { someBackendUtil } from '@shared/backend/utils';
import { BackendType } from '@shared/backend/types';
```
